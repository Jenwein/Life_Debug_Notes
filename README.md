# Life_Debug_Notes
调试人生,记录一切

Life Knowledge Base

一个持续更新的人生经验库，用于记录我在学习、工作、生活、情感、人际等各领域遇到的问题、反思和解决方案。希望通过结构化积累，让自己的成长路径更清晰、更可复用。

📌 仓库目的

系统化记录人生中出现的各种问题

复盘与纠错，形成个人方法论

积累经验，为未来的自己提供参考

构建一个长期可维护的知识体系

📂 目录结构
/
├── learning/        # 学习方法、技能提升、学科知识
├── career/          # 职业发展、沟通、工作经验
├── life/            # 生活管理、习惯、健康
├── emotion/         # 情绪、人际关系、心理认知
├── philosophy/      # 思维方式、价值观、决策原则
├── problems/        # 按时间记录的具体问题与反思
└── templates/       # 标准化记录模板


各目录会随着成长不断扩展与调整。

🧭 问题索引（problems/）

所有具体问题按照时间创建，比如：

problems/
└── 2025-01-learning-plateau.md


文件名格式为：

YYYY-MM-短标题.md


用于追踪问题出现的背景与解决路径。

🧩 记录模板（templates/）
```shell
# 问题标题
## 背景
## 触发原因
## 我的错误认知
## 分析（逻辑纠正）
## 解决方案
## 后续观察
## Tag:
```

便于保持高质量的一致记录。

🔧 使用方式

遇到问题 → 在 problems/ 中创建新记录

复盘之后 → 将总结迁移至对应主题目录

定期梳理 → 更新人生方法论与长期策略

不断迭代 → 使仓库成为自己的“人生知识库”

⭐ 目标

让人生中的每次挫折、困惑和突破都能留下痕迹，最终形成一个值得终身维护的成长体系。
